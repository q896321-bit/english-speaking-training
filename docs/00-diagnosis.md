# 起點診斷

## 當前狀態 (2026-05-31)

- TOEIC 450 (Listening 275 / Reading 175)
- 單字量 2000-3000 (passive recognition only)
- 學習年數: 10+ 年（國高中 + 大學 + 碩士）
- 主要學習模式: 閱讀 / 看影集 / 學校考試
- **核心問題: 無法在腦中構造英文句子**

---

## 數據解讀

### L275 > R175 的異常比例

一般台灣考生 R > L（閱讀有時間思考、有翻譯本能）。你的反向比例代表：

1. **聽力 275 並非真實理解**——是「碎片化模式辨識」。靠關鍵字 + 語調猜意思，不是逐句解碼。
2. **閱讀 175 暴露真正能力**——文法解析速度太慢。論文等級長句會卡在每個從屬子句邊界。
3. **產出失敗是必然**：產出需要 (a) 即時語法規劃 + (b) 詞彙檢索並行。(a) 不夠自動化 → 工作記憶被吃光 → 大腦當機。

### 核心病灶: 翻譯管線

```
你目前：  中文意圖 → 翻成英文 → 文法檢查 → 發音 → 輸出  (3~5 秒/句)
母語者：  英文意圖 → 輸出                              (<0.5 秒/句)
```

這是兩條不同的神經迴路。10+ 年的閱讀訓練了「英→中」高速公路，但「英直接產出」根本沒鋪路。

### 隱性瓶頸: 韻律 (Prosody)

台灣學習者過度執著個別發音。真正關鍵是 **prosody**: 重音、連音、語調起伏。
- 韻律對 + 每個字念錯 → 聽得懂
- 韻律錯 + 每個字念對 → 聽不懂

Shadowing 自然修正韻律，因為你模仿的是「聲音的波形」不是字。

## 為什麼傳統方法行不通

| 方法 | 為什麼失敗 |
|---|---|
| 背更多單字 | 你不缺單字，缺自動化 |
| 學更多文法 | 已經知道，但用不出來（declarative ≠ automatic） |
| 看更多影集 | 純輸入，沒輸出 |
| 偶爾找人聊天 | 量太少（一次 1 hr → 累積太慢） + affective filter 太高（焦慮 → 學習中斷） |
| App 練口說 | 通常是發音 drill，沒練「即時構句」 |

---

## 理論基礎（5 個關鍵假設）

1. **Output Hypothesis (Swain 1985)**: 強制產出會暴露語言缺口（noticing），這個 noticing 才是學習觸發點。
2. **Skill Acquisition Theory (DeKeyser 2007)**: declarative → procedural → automatic。你卡在第一階段。需要刻意練習推到 automatic。
3. **Affective Filter Hypothesis (Krashen 1985)**: 焦慮直接阻斷習得。低壓環境是必需，不是 nice-to-have。
4. **Lexical Approach (Lewis 1993)**: 語言是 chunks，不是 grammar + words。母語者口語 ~50% 是預製句塊。
5. **Spaced Repetition (Ebbinghaus)**: 每日 15 min × 100 天 >> 1 次 25 hr 集訓。

---

## 解決方案的方向

訓練必須有 4 個特徵：

1. **強制產出** (Output Hypothesis)
2. **高頻自動化** (Skill Acquisition Theory)
3. **低焦慮環境** (Affective Filter)
4. **每日小量** (Spaced Practice)
