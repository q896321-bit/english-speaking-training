# Practice Logs

每日練習記錄。

## 為什麼要寫 log

- **記錄 > 感覺**: 你「覺得有沒有進步」永遠不準
- **連續性可視化**: 看著日期 streak 增長有複利效應
- **回顧靠 log**: 半年後你不會記得 Week 3 卡在什麼，log 會記得

## 格式

每天 1 個檔案: `YYYY-MM-DD.md`

簡短即可，3 行也行：

```markdown
# 2026-06-15

- Shadowing: 6min BBC, OK 韻律
- Anki: 5 new, 12 review, 88%
- Self-talk: 8min on weekend plans, 卡 "我覺得受夠了" → got "I've had enough"
- 朗讀: paper section 2.1

問題: forgetting "It depends" register
明天: 用 academic-50 #15 motivated by 開頭寫一段論文
```

## 每週

週日複製 [`../drills/weekly-template.md`](../drills/weekly-template.md) → 存為 `week-XX.md`
