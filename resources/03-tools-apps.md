# Tools & Apps — 你會用到的工具

按優先順序排列。**全部免費 tier 都夠用**。

---

## 🔥 必裝（沒這些不能開始）

### 1. Anki (Spaced Repetition System)
- **桌面**: https://apps.ankiweb.net/ (免費)
- **Android**: AnkiDroid (免費)
- **iOS**: AnkiMobile ($25 一次性)
- **Web**: https://ankiweb.net (免費，但功能少)

**為什麼必裝**: 你不能靠手寫卡片管理 250+ chunks。Anki 用 SRS 演算法精準推送該複習的卡片。

**初次設定**:
1. 建立 deck: `english-speaking-training`
2. 子 deck: `daily-80`, `academic-50`, `business-50`
3. 開啟 AnkiWeb sync（手機↔電腦）
4. 卡片型態: Basic (front + back)

**每日設定**:
- New cards per day: **5** (不要貪心)
- Reviews per day: 不限
- Learning steps: 1m 10m
- Graduating interval: 1 day

### 2. 手機錄音 App
- 系統內建即可
- **用途**: self-talk 錄音、每週重聽
- **進階**: Voice Memos (iOS) / Easy Voice Recorder (Android)

### 3. Python + Anthropic SDK
- Python 3.10+ — https://www.python.org/downloads/
- `pip install anthropic`
- API Key 從 https://console.anthropic.com/

---

## ⭐ 強烈建議

### 4. ELSA Speak (AI 發音教練)
- https://elsaspeak.com/
- 免費 tier 每天 10 分鐘
- 用途: 修個別單字發音
- **注意**: ELSA 練 pronunciation, shadowing 練 prosody — **兩個不可互替**

### 5. Linguee (雙語例句)
- https://www.linguee.com/
- Chrome 擴充: 選文字按右鍵就查
- 用途: 看 native 怎麼用一個字

### 6. YouGlish
- https://youglish.com/
- 輸入英文字 → 跳到 YouTube 中 native 講這個字的片段
- 用途: 聽各種口音怎麼念這個字

### 7. VoiceTube
- https://tw.voicetube.com/
- iOS / Android app
- 中英字幕影片、可重複跟讀

## 💰 Stage 2 後考慮付費

### iTalki (真人 tutor)
- https://www.italki.com/
- **Community Tutor**: $10-15/hr（推薦：便宜，多練口說）
- **Professional Teacher**: $20-40/hr
- **建議頻率**: Stage 2 中期 → 2x/week, 30 min/次
- **如何挑 tutor**:
  - 看試聽影片
  - 從便宜的開始
  - 鎖定 1-2 個 tutor 建立默契

### Cambly
- https://www.cambly.com/
- 隨叫隨到、較貴
- 適合 Stage 3 找 native 練商務情境

### TOEIC 模擬考
- 官方付費題本: TOEIC Official Test-Preparation Guide
- **每季買 1 本**做完整模擬考（追蹤進度）

## 🛠 其他輔助

### 字典 / 查詢
| 工具 | 用途 |
|---|---|
| Cambridge Dictionary | 主要字典，含 collocations |
| Merriam-Webster | 美式字典 |
| Longman Dictionary of Contemporary English | 例句多 |
| Linguee | 雙語例句 |
| Ludwig | 英文搭配查詢 |
| YouGlish | 聽 native 發音 |
| Reverso Context | 翻譯 + 例句 |

### 文法檢查
- **Grammarly** (https://www.grammarly.com/) — 免費版夠用
- **LanguageTool** (https://languagetool.org/) — 開源替代
- **Claude / ChatGPT** — 給段落請它指出 3 個問題

### 寫作練習
- **Hemingway Editor** (https://hemingwayapp.com/) — 簡化句子
- **750words.com** — 每日 750 字練習（可用英文）

### 翻譯 (謹慎使用)
- DeepL (https://www.deepl.com/) — 比 Google Translate 強
- **警告**: 學習階段不要當拐杖。會強化「中翻英」管線。**只在 self-talk 結束後查不會的詞**。

---

## 🚫 不推薦 / 避雷

- **Duolingo** — 適合零基礎啟蒙，但你已經過了那階段，繼續用是浪費時間
- **Rosetta Stone** — 太貴且效率不高
- **單字 App 一堆背單字** — 你不缺單字，缺 chunks
- **YouTube 上「30 天學會英文」之類影片** — 沒有 30 天的奇蹟，只有 600 小時的累積

## 安裝順序建議

**Day 0**（必裝）:
1. Anki Desktop + AnkiDroid/AnkiMobile + AnkiWeb 帳號
2. Python + anthropic SDK
3. Anthropic API key
4. ELSA Speak

**Week 1 內補**:
5. Linguee Chrome 擴充
6. YouTube 「English Shadowing」播放清單
7. Spotify 訂閱 podcast

**Stage 2 開始時**:
8. iTalki 註冊 + 試聽 3 個 tutor
9. TOEIC 官方題本

**Stage 3 開始時**:
10. Grammarly 帳號
11. 訂閱 1 個商業 podcast
